线性代数——非齐次线性方程组、

非齐次线性方程组

Ax=β (β≠0)

1、解的情况：

​ ①无解： r(A) ≠ r( A | β)

​ ②有解： r(A) = r( A | β) ：唯一解：r(A) = r( A | β ) = n；无穷解：r(A) = r( A | β ) < n

2、求解：

​ ①通解 = 齐次通解+非齐次特解

​ ②根据自由变量个数

AB = 0 可知：1° B的列向量为Ax=0的解；2° r(B) ≤ n - r(A) 即 r(B)+r(A) ≤ n

若非齐次存在n个无关的解，则齐次至少存在n-1个无关的解。

公共解、同解

齐次线性方程组Ax＝0与 Bx＝0的公共解的求法
​ （1）联立求解，其解即为公共解（齐次一定有公共解——零解）；
​ （2）由Ax＝0与Bx＝0的通解表达式相等变为关于通解中任意常数的方程组，求通解；
​ （3）将Ax＝0的通解表达式代入Bx＝0中，再确定其通解表达式中任意常数应满足的条件，即得非零公共解。

政治基础——马原

社会存在（物质）和社会意识（意识）：

  • 社会存在包括：生产及生产方式（生产力+生产关系）决定性、地理环境、人口因素重要因素
  • 社会意识包括：社会意识的各种形式+社会心理与自发形成的风俗、习惯
  • 形式分为形态（与阶级有关）（包括法律、经济、艺术、道德……）和非形态（与阶级无关）（包括自然科学等）
  • 二者关系中重要的是：1、社会存在决定社会意识，社会意识是社会存在的反应并反作用于社会存在；2、随着社会存在的发展、社会意识也相应地或迟或早地发生变化和发展。3、社会意识与社会存在发展地不完全同步性和不平衡性

人民群众：

  • 一切对社会历史发展起推动作用地人们+社会人口中的绝大多数+最稳定的主体部分始终是从事物质资料生产的劳动群众及其知识分子。
  • 人民群众是历史的主体，是历史的创造者。
  • 在社会历史发展过程中，人民群众起着决定性的作用。

政治经济学：简单商品经济（资本主义之前，围绕价值）+高度发达的商品经济（资本社会及其以后，围绕剩余价值）

  • 价值是什么？——劳动
    • 商品具有使用价值（自然属性，一切劳动产品地共有属性）和价值（商品特有的社会属性）两个因素。
    • 商品的价值和使用价值是对立统一的！二者不可兼得，二者缺一不可。
  • 衡量价值——社会必要劳动时间；以简单劳动为尺度计量
    • 劳动生产率：社会（劳动生产）率提高，则单品价值量下降；个人率提高，则单品价值量不变！
  • 价值的表现：四个阶段
    • 货币的五种基本职能：价值尺度，流通手段，贮藏手段，支付手段，世界货币
    • 纸币不能执行价值尺度地功能。
  • 价值规律：内容3条；表现形式是商品的价格围绕价值自发波动。因素——供求、币值、价值（决定）。
    • 价格规律地作用：正反两方面。
